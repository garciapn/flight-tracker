# Contributing to Flight Tracker

Thanks for your interest! Here's how you can help.

## Getting Started

1. Fork the repo
2. Clone locally
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Test locally
6. Commit with clear messages
7. Push and open a PR

## Development Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

## Code Style

- Keep it simple and readable
- Use descriptive variable names
- Comment complex logic
- Test before committing

## Ideas for Contributions

- **Multi-route support** - Add tracking for other airport pairs
- **Price prediction** - ML model for predicting prices
- **Email alerts** - Send notifications when deals drop
- **Mobile app** - React Native version
- **Comparison features** - Compare with other booking sites
- **Bug fixes** - Found an issue? Let's fix it

## Reporting Issues

Include:
- What you were trying to do
- What happened
- Expected behavior
- Steps to reproduce
- Python/Flask versions

## Questions?

Open an issue or start a discussion. Happy to help!
